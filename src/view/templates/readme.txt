1) colorTemplate.php - код файла color.h, используется в примерах индикационных кадров.
2) datTemplat.php - код файла dat.h, используется в примерах индикационных кадров.
3) descriptionTemplate - шаблон страницы для описания функции из библиотеки. Состоит из следующих разделов:
описание, синтаксис, параметры, графическое отображение, пример.
4) exampleListMainPage - чать страницы MainPage, вынесенная в отдельный файл. Это сделано для удобства,
чтобы проще ориентироваться в самом файле MainPage.
5) exampleMain.php - стартовая страница для раздела примеров. Состоит из заголовка и краткого описания
раздела.
6) exmapleTemplate.php - шаблон страницы для описания различных примеров. Содежит описание .h и .cpp файла,
а также шаблон для конкретного примера использования ( сам пример использования вынесен в отдельный файл ).
7) libraryMain.php - стартовая страница для раздела библиотеки. Состоит из заголовка и кратокого описания раздела.
8) MainPage.php - главная стрпница, содержит всю основную разметку. На ней полностью реализован хэдер, а также поисковая строка сайта.
Остальная реализается вынесена в отдельные файлы.
9) subExampleContent.php - шаблон страницы, которая является дополнением к exmapleTemplate.php. Содержит в себе
разметку для примера, а также color.h и dat.h
